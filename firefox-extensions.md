# Firefox extensions

These are all the Firefox extensions I use.

- [ClearURLs by Kevin R.](https://addons.mozilla.org/en-US/firefox/addon/clearurls/)
- [Cookie AutoDelete by CAD Team](https://addons.mozilla.org/en-US/firefox/addon/cookie-autodelete/)
- [Decentraleyes by Thomas Rientjes](https://addons.mozilla.org/en-US/firefox/addon/decentraleyes/)
- [DuckDuckGo Privacy Essentials by DuckDuckGo](https://addons.mozilla.org/en-US/firefox/addon/duckduckgo-for-firefox/)
- [HTTPS Everywhere by EFF Technologists](https://addons.mozilla.org/en-US/firefox/addon/https-everywhere/)
- [new-tab by CasperTheGhost](https://addons.mozilla.org/en-US/firefox/addon/new_tab1/)
- [OctoLinker by Stefan Buck, Joseph Frazier](https://addons.mozilla.org/en-US/firefox/addon/octolinker/)
- [Privacy Badger by EFF Technologists](https://addons.mozilla.org/en-US/firefox/addon/privacy-badger17/)
- [React Developer Tools by React](https://addons.mozilla.org/en-US/firefox/addon/react-devtools/)
- [Redux DevTools by Firefox user 12419617, Jhen, timdorr](https://addons.mozilla.org/en-US/firefox/addon/reduxdevtools/)
- [Refined GitHub by Sindre Sorhus, Fregante](https://addons.mozilla.org/en-US/firefox/addon/refined-github-/)
- [Smartschool Dark Mode by Van de Voorde Matthias](https://addons.mozilla.org/en-US/firefox/addon/smartschool-dark-mode/)
- [uBlock Origin by Raymond Hill](https://addons.mozilla.org/en-US/firefox/addon/ublock-origin/)
- [Violentmonkey by Gerald](https://addons.mozilla.org/en-US/firefox/addon/violentmonkey/)
- [WhatFont by Zack Michener](https://addons.mozilla.org/en-US/firefox/addon/zjm-whatfont/)
